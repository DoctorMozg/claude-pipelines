# mozg-pipelines

A public plugin repository for [Claude Code](https://claude.com/claude-code). Import agents and skills into your Claude Code setup.

## Installation

```bash
claude plugin marketplace add DoctorMozg/claude-pipelines
```

Then install individual plugins:

```bash
claude plugin install mz-dev-base
```

## Available Plugins

| Plugin        | Description                                                                            | Version |
| ------------- | -------------------------------------------------------------------------------------- | ------- |
| `mz-dev-base` | Essential development agents, skills, and context-aware rules for coding workflows     | 0.1.0   |
| `mz-dev-pipe` | Full development pipeline — autonomous coding, reviewing, testing, and research agents | 0.1.0   |
